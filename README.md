# Libro
Данный репозиторий предназначен для хранения *исходного кода* приложения домашней библиотеки "Libro"

 **«Libro»** — это мобильное приложение, предназначенное для управления личной коллекцией книг. Цель проекта — перенести традиционную домашнюю библиотеку в цифровое пространство, предоставив пользователю удобный, структурированный и функциональный инструмент для каталогизации, отслеживания и анализа своего чтения.

**Функциональные возможности:**
- Учет книжной коллекции: пользователь может добавлять книги вручную с указанием названия, автора, года издания и других ключевых параметров или при помощи сканирования штрих-кода.
- Категоризация библиотеки: книги систематизируются по виртуальным книжным шкафам (например: гостиная, спальня, кабинет), что помогает организовать пространство и быстро находить нужные издания.
- Отслеживание прогресса чтения: реализована система статусов («хочу прочитать», «читаю», «прочитана», «отложена»), позволяющая контролировать процесс ознакомления с литературой.
- Визуализация статистики: данные о читательской активности отображаются в виде графиков и диаграмм, что упрощает анализ литературных предпочтений и позволяет оценить динамику чтения.
- Гибкая система поиска и фильтрации: пользователь может быстро находить книги по различным критериям - автору, жанру, статусу чтения, расположению в шкафу.
- Система достижений: механика с наградами за различные читательские достижения ("Первый читатель", "Марафонец", "Книжный червь" и др.), которая мотивирует пользователя к регулярному чтению и пополнению библиотеки.
